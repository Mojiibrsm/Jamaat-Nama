# **App Name**: Sangbad Mimangsha

## Core Features:

- Article Summarization: Uses a generative AI model to summarize articles, presenting users with key information quickly. Acts as a tool to assist the user.
- News Feed: Display a scrollable list of news articles with titles, summaries, and publication dates.
- Category Filtering: Allow users to filter news articles by category.
- Search Functionality: Implement a search bar for users to find specific news articles.
- Article View: Provide a detailed view for each article with the full content and images.

## Style Guidelines:

- Primary color: Dark crimson (#8B0000), for a serious, authoritative feel appropriate to news.
- Background color: Very light grey (#F0F0F0) providing a clean, readable backdrop.
- Accent color: Dark maroon (#800000), to highlight important sections and interactive elements.
- Headline font: 'Playfair' serif for headlines, providing an elegant, high-end feel.
- Body font: 'PT Sans' sans-serif for body text, creating a readable and modern user experience.
- Simple, clear icons for categories and actions.
- Clean, organized layout to prioritize readability.
